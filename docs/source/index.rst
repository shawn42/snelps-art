Gumo documentation
==================

*Gumo* is an art set based upon the game design document for
`Snelps <https://github.com/shawn42/snelps>`_, a comical real-time strategy
game. Hopefully these pages will describe the art set in detail and give proper
directions on how to use it.

| `Issue tracking system <https://bitbucket.org/introsmedia/gumo/issues>`_
| `Source files <https://bitbucket.org/introsmedia/gumo/src>`_
